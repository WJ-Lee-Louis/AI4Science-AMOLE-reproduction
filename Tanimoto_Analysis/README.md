# Tanimoto top-100 analysis

This directory separately analyzes all 1,249,975,000 unique molecule pairs and the 5,000,000 saved self-excluded top-100 similarities. Histogram bins have width 0.05. Case-selection criteria and exact molecule metadata are recorded in `analysis_summary.json`; all plot values are also exported as CSV. The top-50 candidate distribution is provided separately with the proposed 0.25 minimum-similarity threshold marked on the plot.
